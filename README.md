# Blog App

This is an application created with Next.js 13

## Install json-server

Install json-server.

```bash
npm i json-server
```

Start server.

```bash
npm run json-server
```

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
